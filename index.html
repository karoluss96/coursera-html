Front-End Languages

HTML5

HTML is core language of web and is important to render data on browser. HTML has a standered version which is now widely accepted by all which is HTML5. It decides basic bone of the content like which will be heading, which will be paragraph, which section will be having which data.

CSS3

To improve visual appereance and to make a webpage attractive, CSS is very important. It makes the look of website beautiful. With Css we can change look, colors, fonts, layout, styles, animations of the webpage completely making it very attractive. It also increases user experience of the product.

Bootstrap

Bootstrap is a CSS framework which is built and maintained by Twitter. It has mobile first philosophy and it is very useful to make responsive website. It also supports rapid development of website because most of the components are already coded on their website.
